Title: Is a given input a valid TeX ⟨number⟩?

Summary: plain TeX conditional-like macro to test for a syntactic ⟨number⟩.

Full text of article: available to TUG members (join TUG); will be publicly available after the next issue is published.

Author: Udo Wermuth

Publication: TUGboat volume 45, number 1 (2024), pages 106–109

DOI (this page): 10.47397/tb/45-1/tb139wermuth-isint
(previous doi - next doi)

Category: Macros

Difficulty: Advanced

Abstract: This article discusses the question of how one can determine if a given string of characters represents a valid number for TeX. A macro that looks and behaves like a Boolean conditional is implemented to answer the question.
