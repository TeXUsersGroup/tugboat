Title: Semi-automated TikZ directed acyclic graphs in R

Summary: brief tutorial on using the causalDisco R package to generate DAGs.

Full text of article: available to TUG members (join TUG); will be publicly available after the next issue is published.

Author: Travis Stenborg

Publication: TUGboat volume 45, number 1 (2024), pages 115–116

DOI (this page): 10.47397/tb/45-1/tb139stenborg-dags
(previous doi - next doi)

Category: Graphics

Difficulty: Intermediate

Abstract: Directed acyclic graphs (DAGs) are a key visualisation tool in graph theory. Semi-automated generation of TikZ code for rendering DAGs is introduced. Automatic TikZ generation via the causalDisco package of the R statistical programming language is proposed. Such easy, rapid DAG generation for LaTeX environments alleviates the need for tedious manual layout of DAG vertices and edges.
