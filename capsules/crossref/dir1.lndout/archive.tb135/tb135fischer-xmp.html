Title: Adding XMP metadata in LaTeX

Summary: LaTeX kernel support for standard metadata.

Full text of article: available to TUG members (join TUG); will be publicly available after the next issue is published.

Authors: Ulrike Fischer, Frank Mittelbach

Publication: TUGboat volume 43, number 3 (2022), pages 263-267

DOI (this page): 10.47397/tb/43-3/tb135fischer-xmp
(previous doi - next doi)

Category: Electronic Documents

Difficulty: Intermediate Plus

Abstract: One task of the ‘LaTeX’ Tagged PDF Project [6] is to evaluate existing solutions to add XMP metadata to a PDF, and if needed, to design and implement a new standard interface for this. In this article we will describe the current state of this task.
