Title: Extracting information from (La)TeX source files

Summary: a generalized Scheme tool for external processing of LaTeX documents.

Full text of article: available to TUG members (join TUG); will be publicly available after the next issue is published.

Author: Jean-Michel Hufflen

Publication: TUGboat volume 43, number 2 (2022), pages 136-141

DOI (this page): 10.47397/tb/43-2/tb134hufflen-extract
(previous doi - next doi)

Category: Software & Tools

Difficulty: Advanced

Abstract: We present some tools that allow us to parse all or part of (La)TeX source files and process suitable information. For example, we can use them to extract some metadata of a document. These tools have been developed in the Scheme functional programming language. Using them requires only basic knowledge of functional programming and Scheme. Besides, these tools could be easily implemented using a strongly typed functional programming language, such as Standard ML or Haskell.
