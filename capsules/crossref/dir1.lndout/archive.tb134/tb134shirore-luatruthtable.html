Title: The luatruthtable LaTeX package

Summary: using Lua to generate truth tables for general logical expressions.

Full text of article: publicly available now.

Authors: Chetan Shirore, Ajit Kumar

Publication: TUGboat volume 43, number 2 (2022), pages 172-175

DOI (this page): 10.47397/tb/43-2/tb134shirore-luatruthtable
(previous doi - next doi)

Category: LaTeX

Difficulty: Intermediate

Abstract: This paper describes the development and usage of the luatruthtable package in LaTeX. It is developed to generate truth tables of boolean values in LaTeX documents. The package provides an easy way of generating truth tables in LaTeX. There is no need of a special LaTeX environment for generation of truth tables with the package. It is written in Lua and the TeX document is to be compiled with the LuaLaTeX engine.
