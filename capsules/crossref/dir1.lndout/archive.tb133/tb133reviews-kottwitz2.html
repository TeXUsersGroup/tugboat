Title: Book reviews: LaTeX Beginner's Guide, second edition, by Stefan Kottwitz

Summary: review of this introduction for today's new LaTeX users.

Full text of article: publicly available now.

Author: Sarah Lang

Publication: TUGboat volume 43, number 1 (2022), pages 73-74

DOI (this page): 10.47397/tb/43-1/tb133reviews-kottwitz2
(previous doi - next doi)

Category: Book Reviews

Section: Reports and notices

Abstract: Kottwitz’s book is a resource for prospective LaTeX users. It is an ideal crash course for self-study. This book gives them all the basics they need to get started and then make an informed decision whether they want to continue their LaTeX journey. For a beginner, this book answers the crucial question of how to get started using LaTeX and thus serves as a much-needed guide to the jungle of resources out there.
