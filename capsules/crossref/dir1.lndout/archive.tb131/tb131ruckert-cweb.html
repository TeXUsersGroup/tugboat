Title: The WEB to CWEB conversion of TeX

Summary: source code to source code conversion: memory words, macros, case statements, and more.

Full text of article: publicly available now.

Author: Martin Ruckert

Publication: TUGboat volume 42, number 2 (2021), pages 139-142

DOI (this page): 10.47397/tb/42-2/tb131ruckert-cweb
(previous doi - next doi)

Category: Software & Tools

Difficulty: Advanced

Abstract: This paper describes several aspects of the conversion of TeX’s source code from WEB, based on Pascal, to CWEB, based on C, with web2w. It emphasizes those aspects that are relevant for obtaining a translation that can truly be regarded as source code and lends itself to modifications.
