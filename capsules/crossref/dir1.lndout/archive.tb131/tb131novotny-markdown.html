Title: Markdown 2.10.0: LaTeX themes & snippets, two flavors of comments, and LuaMetaTeX

Summary: styling Markdown in LaTeX, literate comments, future projects.

Full text of article: publicly available now.

Author: Vít Novotný

Publication: TUGboat volume 42, number 2 (2021), pages 186-193

DOI (this page): 10.47397/tb/42-2/tb131novotny-markdown

Category: LaTeX

Difficulty: Intermediate

Abstract: Celebrating its fifth birthday, the Markdown package has received five new features: user-defined LaTeX themes, LaTeX setup snippets, semantic HTML comments, lexical TeX comments, and support for the LuaMetaTeX engine. In this article, we introduce each of these features and show how they can be used in practice. We also discuss five ideas for the future of Markdown and show how you can help turn them into a reality.
