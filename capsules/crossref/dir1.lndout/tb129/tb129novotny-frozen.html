Title: Making Markdown into a microwave meal

Summary: freezing Markdown, Minted, and BibLaTeX output for single-run processing.

Full text of article: available to TUG members (join TUG); will be publicly available after the next issue is published.

Author: Vít Novotný

Publication: TUGboat volume 41, number 3 (2020), pages 318-320

DOI: 10.47397/tb/41-3/tb129novotny-frozen
(previous doi - next doi)

Category: LaTeX

Difficulty: Intermediate Plus

Abstract: In today’s academic publishing, many venues request LaTeX source in addition to or instead of PDF documents. This is often for the purpose of editing and improving full-text search. Services such as arXiv, Editorial Manager, and EasyChair require LaTeX source code that can be microwaved in a single run of pdfTeX without shell access and without invoking external programs. This requires that authors include auxiliary files and limits their selection of LaTeX packages. In this article, I will show how LaTeX documents using the Markdown, Minted, and BibLaTeX packages can be precooked and frozen to be later microwaved in a single run of pdfTeX.
