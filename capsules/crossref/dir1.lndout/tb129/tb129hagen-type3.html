Title: User-defined Type 3 fonts in LuaTeX

Summary: constructing a font on the fly that can reference images, other fonts, graphics.

Full text of article: available to TUG members (join TUG); will be publicly available after the next issue is published.

Author: Hans Hagen

Publication: TUGboat volume 41, number 3 (2020), pages 320-323

DOI: 10.47397/tb/41-3/tb129hagen-type3

Category: Software & Tools

Difficulty: Advanced

Abstract: This article describes the generic mechanism present in LuaTeX 1.13 and later to deal with user-defined Type 3 fonts. In a TeX environment Type 3 fonts are normally used for bitmap (PK) fonts. However, they can be useful for other purposes too.
