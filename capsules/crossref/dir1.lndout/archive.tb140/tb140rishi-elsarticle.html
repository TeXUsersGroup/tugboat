Title: Navigating common challenges in manuscript submission: Insights for authors and publishers using Elsarticle and CAS packages

Summary: features and benefits of using standard publisher classes, with Elsevier as the case study.

Full text of article: available to TUG members (join TUG); will be publicly available after the next issue is published.

Authors: Rishikesan T, Apu V, Rajagopal CV, Radhakrishnan CV

Publication: TUGboat volume 45, number 2 (2024), pages 253–256

DOI (this page): 10.47397/tb/45-2/tb140rishi-elsarticle
(previous doi - next doi)

Category: LaTeX

Difficulty: Introductory

Abstract: The elsarticle.cls, cas-sc.cls, and cas-dc.cls classes are the preferred LaTeX class files recommended by Elsevier, a publishing company specializing in scientific, technical, and medical (STM) content for manuscript submissions to their journals. These class files are available on CTAN along with templates, BibTeX style files, and user documentations. In this article, we will mainly explore the purpose, features and benefits of these class files and how to utilize these packages effectively for their intended purposes. We also list the challenges resulting from not using these recommended templates.
