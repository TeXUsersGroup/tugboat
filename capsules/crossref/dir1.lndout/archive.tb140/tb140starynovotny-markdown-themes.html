Title: Markdown themes in practice

Summary: tailor presentations of YAML and Markdown, with examples in cross-format expl3.

Full text of article: available to TUG members (join TUG); will be publicly available after the next issue is published.

Author: Vít Starý Novotný

Category: Software & Tools

Difficulty: Advanced

Abstract: The Markdown package for TeX supports themes that allow TeXnicians to tailor the presentation of Markdown and YAML content on the page. In this article, I will show the current state of Markdown themes using the example of LaTeX templates that I developed for the International Software Testing Qualifications Board (ISTQB). Readers will leave with actionable steps to create or modify Markdown themes for LaTeX, and insights into extending these principles to other TeX engines.
